# GHOSTROOT

A multi-agent archaeology of language: reconstructing a lost proto-tongue from fragments, echoes, and scholarly disagreement.

## Requirements
- Python 3.11+
- (Optional) Ollama for local models

## Setup (uv)
uv venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
uv pip install -e .

## Run
ghostroot
